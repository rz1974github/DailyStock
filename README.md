# 台股看板

這是一個台灣股票實時報價看板網頁，支援自動更新。

## 使用方式

1. 在專案目錄執行：

```bash
node server.js
```

2. 在瀏覽器打開 `http://localhost:3000`
3. 編輯股票代號欄位，使用逗號分隔，例如 `2330,2454,2317`
4. 按「立即更新」查看最新報價

## 支援的股票

預設範例：
- 2330 - 台積電 (TSMC)
- 2454 - 聯發科 (MediaTek)
- 2317 - 鴻海 (Foxconn)

及其他台灣上市公司股票代號

## 注意事項

- 資料來源為 Yahoo Taiwan Stock
- 請務必啟動本機伺服器才能正常運作
- 網頁會自動儲存你最後輸入的股票代號
