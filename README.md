# 台股即時看板 (DailyStock)

這是一個輕量、極簡的台灣股票即時報價看板網頁。採用 Node.js 作為本地代理伺服器，前端為單頁應用程式 (SPA)。

## 核心特色

*   **全市場支援**：透過 Fugle Market Data API，支援**上市 (tse)**、**上櫃 (otc)** 及 **興櫃 (esb)** 股票的即時報價。
*   **自動化管理**：
    *   輸入代號即可自動加入/隱藏股票，免手動維護清單。
    *   股票列表自動依代號數字排序。
    *   自動於交易時間內（13:32 前）每 12 秒更新一次報價，收盤後自動停止更新以節省 API 流量。
*   **實用輔助功能**：
    *   附帶產業分類標籤（自動快取自 FinMind API）。
    *   點擊股票名稱旁的 🔗 可直接開啟 Goodinfo 公司簡介頁面。
    *   可自訂「現存股數」與「交易記錄」，所有資料與清單狀態皆會自動保存在瀏覽器的 `localStorage` 中。

## 安裝與設定

1.  **取得 Fugle API Key**：
    請至 [Fugle 開發者中心](https://developer.fugle.tw/) 註冊並取得您的 API Key。

2.  **設定環境變數**：
    在專案根目錄建立一個 `.env` 檔案，並填寫您的 API Key：
    ```env
    FUGLE_API_KEY=你的_FUGLE_API_KEY_放在這裡
    ```

## 啟動方式

本專案使用 Node.js 原生模組，**無須安裝任何第三方套件** (如 `npm install`)。

1.  在專案目錄執行：
    ```bash
    node server.js
    ```
    *(若有 Port 佔用問題，可使用 `lsof -ti :3000 | xargs kill -9; node server.js` 強制重啟)*

2.  打開瀏覽器前往：`http://localhost:3000`

## 使用說明

1.  **新增股票**：在左上角輸入股票代號（支援逗號分隔，如 `2330, 2454, 7886`），按下「Enter」或「立即新增」按鈕即可。若輸入已存在的代號，則會將其隱藏。
2.  **紀錄資料**：直接在表格中的「現存股數」或「交易記錄」欄位輸入文字，系統會自動儲存。

## 注意事項

*   請將 `.env` 加入 `.gitignore`，避免將私鑰上傳至公開的程式碼代管平台（例如 GitHub）。
*   Fugle API 免費方案有請求次數限制，建議於 Fugle 後台**關閉 Enable Overages**，避免超出額度時被收取額外費用。
